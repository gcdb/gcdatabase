10. New Artifact Card System

Artifact system will be added to boost team's stats. When equipping teams with artifact cards, the whole team will receive the boosted stats. If there are set effects for the artifacts, an additional set effect will be activated.

If you obtain a dupe artifact card, it will turn into [Plate Card]. Plate card will be stored in Inventory > Material. Plate Card can be used as artifact enhancing material.

Card Information:
Card rarity: C~UR
Card type: Character, Monster, Story, Background, Special
