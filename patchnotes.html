■ System Changes and Improvements ■

1. Boar Hat Appearance Changes
After clearing Chapter 13 Episode 231, you will be able to change the apperance of mama Hawk's Boar Hat in world map

2. Registering Favourite Characters
You will be able to register favourite characters and they will be displayed towards the top of your box

3. Patrol Missions added
Patrol Missions are added for chapter 10, 11 and 12

4. Regular Patrol Function
There will be a gauge for patrol that will fill up regardless of logins

5. Gacha Rate Display
Gacha rate will be displayed up to 4 decimal places

6. Death Match Invitation Changes
There is now an option to toggle of Death Match invitations for the entire day

7. Cook a Dish changes
You will now be able to immediately max the number of dish cookable, and when using food to up the intimacy of characters, there's now a MAX button for intimacy

8. Chat System Changes
1. You can now view the profiles of players in World Chat and Knighthood chat
2. You will now be able to mention players by touching [@Nickanme] in chat
Guild leaders/vice leaders will be able to @all to message everyone in guild
3. You will now be able to block messages from certain players
4. World Chat and Knighthood chat player play info update
You will get notified in the chat for players pulling SSR units, Knighthood Boss ranking update, PVP rank up etc, this is toggleable
5. Knighthood Battle related notifications
6. Checking of Knighthood member login status is now added in chat
7. You can now invite Knighthood or world players in chat to Death match

9. Skip Ticket Changes
1. Using New Year costume on the team will now spawn the respective Death Matches when using skip tickets
2. Using skip tickets will now level up characters, player rank and gain affections accordingly

10. [Achivement] related system changes
Certain achievements required to have specific amount of materials previously, after the changes you will be able to get the achievement by having more than what's required

11. Hiding Menu
You will now be able to hide some menu icons

12. Knighthood Battle
Fixed some stuff with that when loading into world map

13. Overall CC Buff
Additional CC buffs are now added (750k, 1.25mil, 175mil)

14. Coin Shop Display
Event Coin Shop items will now be marked with [EVENT]
