■ [Sweet Temptation] Guardian Elaine Skills Info (rank 3)

Link to Jenna Ult

■ SSR [Sweet Temptation] Druid Jenna Skills Info (rank 3)

Skill 1 - Flash: Inflict 250% of this unit's ATK to all enemies. Infect the targets for 3 turns.

Skill 2 - Bright Shot: Remove all stances from a single enemy, and inflict 500% of this unit's ATK to the target.

Ult - Outset: Collapse on a single enemy with 560% of this unit's ATK.

Passive - Disparagement: This unit can only receive 1 debuff effect. When this unit has a debuff, increase the unit's ATK by 20%.

Link to Zanelli Ult

■ SSR [Sweet Temptation] Druid Zanelli Skills Info (rank 3)

Skill 1 - Nimbus: Inflict 450% of this unit's ATK to a single enemy. The target will not be able to gain any ultimate gauge for 2 turns.

Skill 2 -Point Light: Cleanse all debuffs on all allies and heal 50% of the targets' missing HP.

Ult -Rising Nature: Create a HP shield for all allies based on 350% of this unit's ATK (GCDB: they did not specify how many turns), and increase the HP-related stats of targets by 25%.

Passive - Those Who Pray: When this unit cleanses debuff on allies, increase the unit's ultimate gauge by 1.
