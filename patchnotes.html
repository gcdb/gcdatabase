■ System Changes ■

1. Nanashi is now engraveable.

2. Added a function to continuously change the basic stats of a piece of equipment.

3. Added a function to change multiple stats of a piece of equipment at the same time.

4. Added Brawl AI Customize Functionality

5. Changed image that displayed shop menu.

6. Quest and Reward box display changes.

7. When item inventory is MAXed, FULL will be displayed above the inventory.

8. Players can check the type of addditional effects that will be added after the equipment awakening by tapping the magnifying glass.

9. Changed the max amount of AP recovery potions that can be used to refill AP so that the max AP will be below 999.

10. Added a Knighthood icon for players who have not joined a Knighthood to create or join one.
