■ 2. Changes to skill usage ruling

・ From now on, even if a character is under inability effect, you will still be able to select and expend the formally unusable skill(s) of that character
You will be able to use the unsable skills of characters that are under inability effects such as [Freeze], [Petrification] and [Stun]. The outcome of the skill usage will be dependent on whether the inability effect is cleansed prior to the skill's activation.

・ If a cleanse is used prior in the same turn, the skill(s) of the character under inability effect will be used per normal

・ If no cleanse was used, no action will be taken and the skill card will be voided (no activation), however, the character will still be able to gain an ultimate gauge

・ The same ruling is applied to attacking/buffing/debuffing/healing/ultimate skill sealing

Example 1: Using skills of a character under inability effect
※ Without cleansing the debuff prior to a skill's activation: the skill card will be skipped (not activated)
※ Without cleansing the debuff prior to an ultimate skill's activation: the ulimate skill will not be activated but instead return to your hand

For example: Escanor is under petrification.
Before Change: No cards can be selected
After change: You will be able to select the skills but it will not be activated

Example 2: Using skills of a character under card sealing effect(s)
※ Without cleansing the debuff prior to a skill's activation: the skill card will be skipped (not activated)

For example: Your team is under attacking card sealing effect.
Before Change: No cards can be selected
After change: You will be able to select the skills but it will not be activated

Example 3: Using a cleanse prior to using skills of a character under inability/card sealing effect(s) in the same turn
The debuff will be cleansed and the skills will be activated normally

For example: Your team is under card sealing effect.
Before Change: No cards can be selected even if a cleanse was used prior
After change: The debuff will be cleansed and the skill(s) will be activated as per usual
